# Η Ακαδημία των Μαθηματικών Ηρώων

Διαδραστική πλατφόρμα Μαθηματικών Α' Γυμνασίου με homepage, κεφάλαια, μάθημα 1.1, XP, quiz και boss challenge.

## GitHub Pages

Το project περιλαμβάνει έτοιμο GitHub Actions workflow στο `.github/workflows/deploy.yml`.

Ανέβασε όλα τα περιεχόμενα στο repository `mathimatika-a-gim` και άφησε το GitHub Actions να κάνει deploy.
