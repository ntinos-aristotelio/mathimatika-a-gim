# Μαθηματικά Α' Γυμνασίου - Ακαδημία Μαθηματικών Ηρώων

Version 3: Πρώτο ολοκληρωμένο μάθημα 1.1.

## Περιλαμβάνει

- Αναλυτικό sidebar με υποενότητες του σχολικού βιβλίου
- Πλήρες μάθημα 1.1: Φυσικοί αριθμοί - Διάταξη - Στρογγυλοποίηση
- Θεωρία, παραδείγματα, quiz και boss challenge
- XP, level, hearts, streak
- Αυτόματο deployment με GitHub Actions

## Τοπική εκτέλεση

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
